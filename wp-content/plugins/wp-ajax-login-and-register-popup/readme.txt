WP AJAX Login and Register Popup is a WordPress plugin to login and register on your website using popup box!

== Description ==
WP AJAX Login and Register Popup is a WordPress plugin to login and register on your website using popup box!  Use this shortcode <strong>[WPALRP-LOGIN]</strong> in the widget where you want to display login/register button.

= Usage =

1. Go to your Dashboard after installation and navigate to "Settings >> AJAX Login Settings" to configure the Login/Register.
2. Go to Widget after installation and drop "AJAX Login or Register" to your desired area, where you want to display AJAX Login or Register button.
3. Add extra ID "show_login" at login menu & ID "show_signup" at register menu.

== Installation ==
1. Upload `wp-ajax-login-register-popup` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Just copy and paste <strong> if(function_exists('wpalrp_popup_functions')){ wpalrp_get_data_from_settings(); } </strong> in template code or  <strong>[WPALRP-LOGIN]</strong> in the post/page where you want to display login/register button.
